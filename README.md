# UCCARON

A modern, interactive text-to-speech web application with a sleek dark theme and smooth animations.

## Features

- **Text-to-Speech**: Convert any text input to natural speech output
- **Real-time Wave Animation**: Visual feedback with animated sound waves while speaking
- **Clean UI**: Modern design with dark theme and custom typography
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Easy to Use**: Simple input field and intuitive controls

##  Live Demo

Visit the application here: **[ornate-lamington-b6e322.netlify.app](ornate-lamington-b6e322.netlify.app)**

## 💻 Getting Started

### Prerequisites
- A modern web browser with Web Speech API support (Chrome, Edge, Firefox, Safari)

### Local Setup
1. Clone this repository
2. Open `Uccaron.html` in your web browser
3. Start typing and click the "Speak" button to hear your text

## 🛠️ Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Custom design with modern styling
- **JavaScript**: Web Speech API integration
- **Fonts**: 
  - Syne (Display font)
  - DM Mono (Body font) from Google Fonts

##  Usage

1. Enter your text in the input field
2. Click the "Speak" button to hear the text read aloud
3. Click "Clear" to reset the input field
4. Customize text as needed

## Design Features

- **Dark Theme**: Easy on the eyes with accent colors (#F0A500, #FF6B35)
- **Animated Wave Icon**: Visual representation of audio output
- **Smooth Transitions**: Polished interactions and hover effects
- **Grid Background**: Subtle texture for visual depth
